# CSS Advanced Project

## Overview
This project continues the HTML Advanced Project, focusing on **CSS styling** to match the Figma design. The goal is to create a clean, responsive, and visually accurate webpage using only HTML and CSS.

## Project Structure
css_advanced/
├── index.html
├── styles.css
└── README.md

- **index.html** – HTML structure of the page.  
- **styles.css** – CSS styling according to Figma design.  

## Features
- Header, banner, quotes, videos list, membership, FAQ, footer sections styled.  
- Responsive design for desktop and mobile.  
- Fonts used: Source Sans Pro, Spin Cycle OT.  
- All CSS is simple, clean, and centralized.
